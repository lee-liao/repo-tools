# Deploy folder for Claude Code Skills

This folder contains the bundled, deployment-ready files for the repo-tools skill.

## Files
- `download_repo.js` - Bundled repository downloader (all dependencies included)
- `get_snippet.js` - Bundled file downloader (all dependencies included)  
- `SKILL.md` - Skill documentation

## Deployment
Copy these files to: `C:\Users\liao_\.claude\skills\repo-tools\`

Or run: `npm run deploy`
